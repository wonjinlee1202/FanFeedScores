# FanFeed Live Scores

**Live Link to Project:** [https://fan-feed-scores.vercel.app/](https://fan-feed-scores.vercel.app/)

Shows live scores from various sport leagues. You can also look at schedules for every team, see team statistics, team standings, and view information about each player. This web app was created using the NextJS React framework. More info on this API can be found [here](https://gist.github.com/akeaswaran/b48b02f1c94f873c6655e7129910fc3b).
